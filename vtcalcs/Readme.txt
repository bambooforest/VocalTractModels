Matlab version of VTCalcs
Version: 1.00

IMPORTANT:
1. The function vtcalcs must be called from the same 
	directory where the program resides. Otherwise, 
	it will not find the appropriate files to read.

INSTALLATION INSTRUCTIONS:
1. Unpack the zip file. Please maintain the directory 
	structure when unpacking.
2.	Start Matlab.
3.	Change directory to where the program was unpacked.
4.	Run vtcalcs


Detailed instructions are provided in doc/index.ps or doc/index.pdf.
